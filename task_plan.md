## 关键决策记录
| 决策 | 选项 | 结论 | 理由 |
|------|------|------|------|
| 交互方式 | 对话/文件分析/全功能 | 全功能平台 | 驱动问题诊断需要多维度输入 |
| 知识来源 | 预置/联网/混合 | 混合模式 | 核心知识预置+联网补充+经验积累 |
| 技术栈 | LangChain/OpenAI/AutoGen | Python + LangChain | 灵活编排工作流，生态成熟 |
| 首期范围 | 全子系统/核心子系统/DTS/Panic | MTK 24E Camera驱动 | 聚焦单一场景做深做透，后续扩展 |
| LLM 默认 | GPT-4/Claude/MiniMax | MiniMax | 用户指定，中文能力强，性价比高 |
| 平台体系 | 单平台/多平台 | 三级多平台(厂商→子平台→项目) | 知识隔离，项目共享平台知识 |
| 知识隔离 | 全共享/按平台隔离 | 按子平台隔离，项目共享 | 避免跨平台知识污染，项目间复用 |
